Production: Minification

You must pass in the arguments as an array.

      app.controller('TestCtrl', ['$scope', function ($scope) {
        $scope.foo = 'Foo';
      }]);

Production: Flash of Angular code

When you refresh the screen, you'll see the {{ foo }} tags before Angular loads.

      <div>{{ foo }}</div>
      <div ng-bind="foo"></div>

Production: Flash of Angular code

When you refresh the screen, you'll see the {{ foo }} tags before Angular loads.

      [ng\:cloak], [ng-cloak], [data-ng-cloak],
      [x-ng-cloak], .ng-cloak, .x-ng-cloak {
        display: none !important;
      }
       
      <div ng-cloak>{{ foo }}</div>

Production: Internet Explorer

  • Angular 1.3 is dropping support for IE8 and below
